📊 India General Elections 2024 – SQL Data Analysis Project
🚀 Project Overview

This project focuses on analyzing the India General Elections 2024 dataset using SQL. The goal was to extract meaningful insights such as seat distribution, party performance, alliance dominance, and voting patterns across constituencies and states.

🛠️ Tools & Technologies
SQL (MSSQL)
Excel (Data Cleaning)
CSV Dataset
📂 Database Design

Designed a relational database schema with the following tables:

constituencywise_results
constituencywise_details
partywise_results
statewise_results
states

🔄 ETL Process
Extracted raw election data from CSV files
Cleaned and transformed data using Excel
Loaded structured data into SQL Server
📌 Key Analysis & SQL Concepts Used

✔️ Aggregations & Grouping
Total seats per state
Party-wise and alliance-wise seat counts

✔️ Joins
Combined multiple tables to derive meaningful insights

✔️ CASE Statements
Categorized political parties into alliances:
NDA
I.N.D.I.A
OTHER

✔️ Window Functions
Ranked candidates using ROW_NUMBER()
Identified winners and runner-ups

✔️ Subqueries & CTEs
Used correlated subqueries and Common Table Expressions for advanced analysis
📊 Key Insights
🏆 Identified which alliance (NDA / I.N.D.I.A / OTHER) won the most seats
📍 Analyzed state-wise seat distribution
🗳️ Compared EVM votes vs Postal votes
🥇 Found top candidates with highest votes
📉 Calculated vote margins and constituency-level performance
🧑‍🤝‍🧑 Identified winning and runner-up candidates per constituency
📈 Sample Business Questions Solved
What is the total number of seats in each state?
Which alliance won the majority of seats?
